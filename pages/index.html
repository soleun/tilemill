### Features

- Access datasources on the local filesystem or use online public datasources from the MapBox library
- Manage map layers with Shapefile, GeoJSON, Raster, PostGIS and other datasources
- Edit `carto` stylesheets directly in the browser or use your own text editor
- Preview map tiles instantly while editing stylesheets
- Inspect datasource field values and data types
- Export maps to PNG, PDF and [MBTiles](http://www.mbtiles.org) formats
